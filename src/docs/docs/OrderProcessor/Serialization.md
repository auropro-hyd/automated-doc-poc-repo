# Serialization

## [IntegrationEventContext](https://github.com/akhileshap9/automated-doc-poc-repo/blob/main/src/OrderProcessor/Extensions/Extensions.cs#L17-L24)

### Overview
IntegrationEventContext is a partial class used for source generation of JSON serialization metadata for integration events, specifically GracePeriodConfirmedIntegrationEvent. It enables efficient and type-safe serialization/deserialization of event payloads in the OrderProcessor service. This class is referenced in configuration to ensure correct serialization settings are applied.
