# eShop.ServiceDefaults Overview

`eShop.ServiceDefaults` is a foundational library for the eShop solution, providing essential extension methods and configuration helpers to streamline service setup, authentication, OpenAPI documentation, HTTP client configuration, and general utility functions. Its primary goal is to standardize and simplify common patterns across microservices, improving maintainability, reliability, and developer productivity.

## Documentation Areas

- [Configuration Extension](./Configuration%20Extension.md): Details configuration helpers and extension methods.
- [OpenAPI Extension](./OpenAPI%20Extension.md): Covers OpenAPI/Swagger setup and customization.
- [HTTP Extension](./HTTP%20Extension.md): Documents HTTP client configuration and authentication token handling.
- [Authentication](./Authentication.md): Explains authentication setup and claims principal helpers.
- [General Utilities](./General%20Utilities.md): Describes general-purpose service setup and utility extensions.

Refer to each section for in-depth documentation, code examples, and onboarding guidance for developers working with `eShop.ServiceDefaults`.
